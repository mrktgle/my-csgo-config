* To install autoexec *
1.Simply go to C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive\csgo\cfg
2.Paste the autoexec.cfg there
3.Done!


* To install the radar edited map and banana gaming font. *
1.Just go to C:\Program Files (x86)\Steam\steamapps\common\Counter-Strike Global Offensive
2.Paste the csgo folder and let it all overwrite the file! 
3.Put a -language bananagaming in your launch options
4.Done!

* MOUSE DPI = 800 + CSGO Mouse Sensitivity = 7.02
